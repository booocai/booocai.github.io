嗝 吃饱一文

发表于 2019-02-22 | 分类于 生活
字数统计: 846 字 | 阅读时长 ≈ 2 分钟

雷猴 这是我手写的第三篇推文
步入2019 第一件大事就是二战失败，因为英语没有过线。虽说好事多磨，可我计划里就没有三战好吧，很烦很烦。自己的计划全部乱了节奏，这一年将会面对乡镇的工作，我爹问我是想好好上班还是划水，我选择了划水。倘若自己不是这个情况，自己也不会喜欢这份工作，何况如今。佳县这个地方，芝麻点大，你可能在街上摔一跤三两天之后整个县城都知道你在哪摔了一跤甚至可以绘声绘色的描绘出摔跤之后那副狼狈的样子。这不是我喜欢的感觉。生活上倒有些喜欢那种性冷淡风。大家各司其职，有啥事按照流程好好走就是了。工作之后没啥感情就不要联系了，自己找找自己朋友喝喝酒，吹吹牛，和对象做爱做的事，把父母照顾好，这不就很好么。我这人相当不喜欢和领导啊不熟的同事打交道，我觉得自己的圈子已经足矣，就是缺个对象一天腻在一起，圈子太大我这人又傻，不知道怎么去经营这种关系，就很烦。不过现在也是感谢这个工作，至少自己做什么事情家里面都是不会阻止的，毕竟使他们眼中一份十分体面地工作。
说到工作，就说说这份工作，其实事业单位这份工作，不吹不黑，只要是个人，都可以做了。只要你用心，就可以做好。其他地方我不知道，就佳县来说，大家是没有人用心去做工作的，都想着怎么靠着这份工作织自己的关系网，即使自己一年赚个几百万，这份工作是不能丢。做什么事情，都是通过自己的关系网去做。你如果自己按照流程办的事本来三五天可以办成的事大概会拖个一两个星期，但是你通过自己的关系网去办，可能要求本人到场什么的所谓硬性规定都不存在，而且，秒秒钟就办出来了。你说，就这样的工作，当然对于喜欢这样生活的人是好的，自己的本事在这里可以大显神通，可是搁到我身上，妈耶，大概就是哭辽的感觉。所以说，这份工作，我选择了划水，作为过渡，也就把自己分内的事情做好就是了，其他的，算是人生阅历了。还要在单位大开眼界吧。hh
现在的生活，大概也就再浪一浪，就去好好学英语了。英语要是学得好，就看之后未来的对象（要是有的话）在哪里念书工作，自己也就屁颠屁颠去好了。念研究生的初衷很简单，一是逃离，二是看看自己专业未来的发展前景，三是满足自己浪的需求吧。对了，还得切学游泳，自己得给自己做个年度计划，学些什么技能，不能在单位待傻了。嘿嘿，溜了，守望爸爸等着我。

杂记 瞎逼逼
